## To start

* 'npm install' 

*  start the development server with `npm start`

## To build

* npm build


## Dependencies

This app was created with the help of ['google-maps-react`](https://www.npmjs.com/package/google-maps-react) and create-react-app.


It makes use of the Wikipedia API.
